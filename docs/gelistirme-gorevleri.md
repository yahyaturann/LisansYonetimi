# Geliştirme Görevleri

## Öncelikli İşler

- [ ] Monorepo klasör yapısını kur
- [ ] Root `package.json` ve workspace yapılandırmasını ekle
- [ ] Backend bağımlılıklarını kur
- [ ] Frontend bağımlılıklarını kur
- [ ] Prisma şemasını oluştur
- [ ] Seed dosyasını hazırla
- [ ] JWT tabanlı admin giriş API’sini yaz
- [ ] Proje oluşturma endpointini yaz
- [ ] Proje ayar güncelleme endpointini yaz
- [ ] Lisans oluşturma endpointini yaz
- [ ] Lisans listeleme endpointini yaz
- [ ] Lisans silme endpointini yaz
- [ ] Log listeleme endpointini yaz
- [ ] Lisans doğrulama endpointini yaz
- [ ] Rate limit ve güvenlik middleware’lerini ekle
- [ ] Backend testlerini yaz
- [ ] Next.js yönetim panelini oluştur
- [ ] Giriş sayfasını bağla
- [ ] Dashboard özet kartlarını bağla
- [ ] Proje yönetimi ekranını oluştur
- [ ] Lisans yönetimi ekranını oluştur
- [ ] Log ekranını oluştur
- [ ] Frontend servis katmanını yaz
- [ ] Ortam değişkenlerini örnekle
- [ ] Build, lint ve test komutlarını doğrula
- [ ] Yerel çalıştırma senaryosunu test et

## Teknik Parçalama

### Backend

- Sunucu girişi
- Prisma istemcisi
- Hata yönetimi
- Kimlik doğrulama middleware’i
- Proje servisleri
- Lisans servisleri
- Doğrulama servisleri
- Log servisi

### Frontend

- Oturum saklama yapısı
- Ortak düzen bileşenleri
- Form bileşenleri
- Veri tabloları
- API istemcisi
- Toast veya bildirim alanı

## Test Hedefleri

- Lisans doğrulama başarılı senaryosu
- Süresi dolmuş lisans senaryosu
- Domain/IP/HWID uyuşmazlık senaryoları
- Aktivasyon limiti senaryosu
- Admin login senaryosu
- Proje oluşturma senaryosu
- Lisans oluşturma ve silme senaryoları

## Tamamlanma Kriteri

Bir görev aşağıdaki koşullar sağlandığında tamamlanmış kabul edilir:

- İlgili kod yazılmış olmalı
- Temel test veya doğrulama komutu çalıştırılmış olmalı
- Kullanıcı arayüzündeki metinler Türkçe olmalı
- Hata ve log mesajları Türkçe olmalı
